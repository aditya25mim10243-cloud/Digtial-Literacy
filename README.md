## 📌 Project Overview
This repository serves as my **Digital Literacy Portfolio**, created in my role as a Student Digital Ambassador. The goal of this project is to demonstrate proficiency in navigating the digital world, from professional branding and coding collaboration to online safety and etiquette.

This project consists of five tasks mapped to the five modules of the course:
1. **Digital Literacy Awareness Infographic** (Module 1)
2. **Student Digital Portfolio** (Module 2)
3. **Coding & Collaboration Platforms** (Module 3)
4. **Professional Email & Etiquette Guide** (Module 4)
5. **Cybercrime Awareness & Prevention** (Module 5)
